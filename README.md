# BORTOLOTO - Site (HTML estático)

Projeto de site estático (HTML/CSS/JS) pronto para GitHub Pages.

## Publicar no GitHub Pages
1. Crie um repositório no GitHub (ex: `bortoloto-site`)
2. Envie todos os arquivos desta pasta para o repositório
3. GitHub: Settings → Pages → Deploy from a branch → branch `main` → / (root)

## Estrutura
- `index.html` (principal)
- `materiais.html`
- `fotos.html`
- `provas.html`
- `modalidades.html`
- `eventos.html`
- `inscricoes.html`
- `aulas.html`
- `assets/` (CSS/JS/imagens)

## Observação
O formulário de inscrições salva dados localmente (LocalStorage). Para receber inscrições online, conecte com Google Forms/Apps Script ou backend.
